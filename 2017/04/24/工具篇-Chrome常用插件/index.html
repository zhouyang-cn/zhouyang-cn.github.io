工具篇---Chrome常用插件

发表于 2017-04-24 |

根据市场调查机构NetMarketShare最新统计数据显示：2016年9月份Chrome浏览器已经以54.41%的占有率居于第一位。

Chrome浏览器以速度快而赢得大家的喜爱，但是对于我们这些程序猿来说，其强大的插件也深深的吸引着我们，今天就来介绍一下Chrome里一些好用的插件。

Adblock Plus ——广告拦截

超过5000万人使用，免费的广告拦截器，可阻止所有烦人的广告及恶意软件和跟踪。

广告终结者

清除网页上的所有广告：浮动广告，购物广告，恶意弹窗，跟踪代码，视频广告。让你浏览网页更快更清爽。

Advanced REST client——http接口测试插件（4星半11936的评论）

The web developers helper program to create and test custom HTTP requests.
一款优秀的api测试工具

Postmam——另一款比较推崇的api测试插件（5星7379的评论）

Supercharge your API workflow with Postman! Build, test, and document your APIs faster. More than a million developers already do.…

Android SDK Search——快速查看android sdk的插件

安装成功后再chrome的地址栏输入ad，然后按tab键输入您要查的关键字按enter键，在地址栏的下方会出现下拉列表，选中你要看的内容。而且界面会生成一个 view source按钮，点此按钮可以查看api对应的源码。

ChromeADB——一款为android开发者使用的插件

支持以下功能：

  • 安装apk
  • 清除数据，卸载apk，强行停止程序
  • 支持返回键，home键，menu键，电源键，音量加减
  • 进程列表
  • app 内存信息
  • 存储空间

JSON Editor——一款展示、编辑和校验Json的工具

支持以下功能：

  • 代码编辑区和树型视图区修改任一边支持同步修改到另一边
  • 支持修改、添加、移除字段和值，并且支持重复字段检测
  • 支持修改值得类型
  • 支持对数组和对象的排序
  • 支持搜索并高亮搜索到的内容
  • 支持撤销和重新执行操作
  • 支持校验json格式

Vysor——一款可以将android手机屏幕投影到电脑的工具

此软件不仅可以投影手机屏幕，还可以通过电脑控制您的手机，比如通过键盘向手机的输入框输入文字，通过鼠标点击手机屏幕，手机屏幕可以响应点击。

最重要的一点就是：速度快，手机上的动画都可以完美展示

Octotree——Code tree for GitHub and GitLab

喜欢上github的朋友相信会喜欢这个插件，可以生成项目的视图树。具体效果见下图

Isometric Contributions——一款可以将github提交记录展现3d效果的插件

废话不多说，直接上JakeWharton大神的profile。

Gliffy Diagrams——一款可以离线使用的绘图工具

可以快速轻松地创建具有专业外观的图表和流程图，下面列举一下它适合的场景：

  • 基本绘图
  • 流程图
  • UML图表
  • 业务流程模型

  • and any more

Speed dial 2——一款new tab插件

很多人不喜欢chrome默认的创建新标签页的效果，这个插件可以优化新标签页的显示效果。
Speed dial 2是有快速访问您最常访问网页的功能，书签和浏览历史记录的终极新标签页替代品。
